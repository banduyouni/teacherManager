# 教学管理系统

一个基于Web的现代化教学管理平台，支持学生、教师、教务管理员和系统管理员四种角色。

## 功能特性

### 🎓 学生功能
- 查看个人信息和课程安排
- 在线选课和退课
- 查看成绩和学业统计
- 个人信息管理
- 消息通知

### 👨‍🏫 教师功能
- 课程管理（创建、编辑、删除）
- 学生管理和成绩录入
- 课表查看
- 消息通知

### 📚 教务管理员功能
- 课程设置和管理
- 学生注册和学籍管理
- 教师管理
- 成绩管理
- 课表编排

### ⚙️ 系统管理员功能
- 系统监控和状态查看
- 用户管理（所有角色）
- 系统设置和维护
- 数据备份和恢复
- 系统日志查看

## 技术架构

### 前端技术
- **HTML5**: 语义化标签和现代Web标准
- **CSS3**: 现代样式设计，响应式布局
- **JavaScript ES6+**: 模块化开发，现代语法
- **CSS Grid & Flexbox**: 响应式布局系统

### 设计特点
- 🎨 现代化UI设计
- 📱 完全响应式，支持移动端
- ♿ 无障碍设计考虑
- 🌈 统一的视觉风格和交互体验

## 项目结构

```
teachManager/
├── index.html              # 登录页面入口
├── login.html              # 登录页面
├── student-dashboard.html  # 学生仪表盘
├── teacher-dashboard.html  # 教师仪表盘
├── academic-admin-dashboard.html  # 教务管理员仪表盘
├── system-admin-dashboard.html    # 系统管理员仪表盘
├── css/
│   ├── common.css         # 通用样式
│   ├── dashboard.css      # 仪表盘基础样式
│   ├── login.css          # 登录页面样式
│   ├── student.css        # 学生页面样式
│   ├── teacher.css        # 教师页面样式
│   ├── academic-admin.css # 教务管理员样式
│   └── system-admin.css   # 系统管理员样式
└── js/
    ├── auth.js           # 认证模块
    ├── dataManager.js    # 数据管理核心
    ├── student.js        # 学生功能
    ├── teacher.js        # 教师功能
    ├── academicAdmin.js  # 教务管理员功能
    └── systemAdmin.js    # 系统管理员功能
```
